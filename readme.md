## Install

	$ git clone https://github.com/ocliuziyang/ocliuziyang.git
	
	$ cd ocliuziyang 
	
	$ composer install

	$ cp .env.example .env

	// 在国内由于国家防火墙的对外网的限制，最好使用阿里巴巴提供的 淘宝 NPM 镜像
	$ npm install
	
	$ npm run dev
	
	//下面可以访问该网站了，你可以使用 valet || homestead || php artisan serv
